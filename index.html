Animation testing tool

This tool was created to debug animation sprite sheets. To test your own image, upload it to any image host and paste the URL into the "Image" box, input the delay between frames in the "Delay" box, and input the height of each frame in the "Height" box.

Notes:

  • This tool supports vertical sprites only. Horizontal or multi-column sprites will not work properly.
  • The 'height' field must be the exact height of each frame. If it's off by even a pixel, the animation will quickly become misaligned.
